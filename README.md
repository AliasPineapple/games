# Games

小游戏合集 —— 寓教于乐，专为小朋友设计。

## 目录结构

```
games/
├── h5/                   # H5 网页游戏（浏览器直接运行）
│   └── hanzi-find/       # 汉字找一找（认字益智）
│       └── index.html
└── wechat/               # 微信小程序游戏（开发中）
```

## H5 游戏列表

| 游戏 | 说明 | 适合年龄 |
|------|------|---------|
| 汉字找一找 | 从字格中找出含目标字的汉字，3关递进难度 | 5-8岁 |

## 运行方式

H5 游戏直接用浏览器打开 `index.html` 即可，无需安装任何依赖。
