# Games

小游戏合集 —— 寓教于乐，专为小朋友设计。

## 目录结构

```
games/
├── design/               # 所有游戏的策划文档（每款游戏一个子目录）
│   └── hanzi-puzzle/     # 汉字拼图消消乐策划案
│       ├── README.md
│       └── design/
│           └── game-design-document.md
├── h5/                   # H5 网页游戏（浏览器直接运行）
│   ├── hanzi-find/       # 汉字找一找（认字益智）
│   ├── hanzi-puzzle/     # 汉字拼图消消乐（开发中）
│   └── gomoku/           # 五子棋
└── wechat/               # 微信小程序游戏（开发中）
```

## 游戏列表

| 游戏 | 状态 | 说明 | 适合年龄 | 策划案 |
|------|------|------|---------|--------|
| 汉字找一找 | 已上线 | 从字格中找出含目标偏旁的汉字，19个年级难度 | 5-18岁 | - |
| 汉字拼图消消乐 | 开发中 | 点选偏旁字根与目标偏旁拼合成汉字，堆叠遮挡玩法 | 6-12岁 | [策划案](design/hanzi-puzzle/design/game-design-document.md) |

## 运行方式

H5 游戏直接用浏览器打开对应目录下的 `index.html` 即可，无需安装任何依赖。

## 新增游戏规范

- 策划文档放 `design/<游戏名>/`
- 工程代码放 `h5/<游戏名>/` 或 `wechat/<游戏名>/`
- 两者通过各自 README 里的相对路径链接互指
